# BodyMap 身体日记

一个以身体地图可视化为核心的个人健康管理 Demo 应用。通过手绘笔记风格的可视化身体地图，帮助用户直观地记录、追踪和理解自己的身体状态。

## 项目介绍

BodyMap 是一款面向移动端的单页健康记录应用（Demo），以"身体地图"为核心理念，将人体各部位的健康状态以彩色圆点的形式呈现在人物 SVG 上，让用户一目了然地看到自己身体的整体状况。

项目采用纯前端实现（HTML + CSS + JavaScript + Chart.js），数据存储于浏览器 localStorage，无需后端服务即可独立运行。当前版本采用手绘笔记（Hand-drawn Notebook）视觉风格，营造出亲切、轻松的健康记录氛围。

## 功能介绍

### 首页 — 身体地图可视化

- **中心辐射式布局**：左侧症状选择面板 + 中央人物身体地图 + 右侧数据便签卡片
- **身体地图圆点**：人体各部位以彩色圆点标识健康状态（绿色=健康、黄色=轻度疲劳、橙色=风险预警、蓝色=正常），支持悬停查看和点击交互
- **症状多选标记**：选择症状（酸痛/紧绷/刺痛/乏力）后，对应颜色的圆点会点亮并发光，点击身体部位即可快速生成记录
- **临时记录管理**：在首页直接选症状+点部位生成临时记录卡片，支持删除和一键提交保存
- **风险警告动画**：肩颈等高风险部位显示警告三角图标，带脉冲动画提醒
- **数据便签卡片**：右侧便签展示久坐时长、建议活动时长等数据
- **底部快捷操作**：记录感受、上传体检、查看趋势三个入口按钮

### 今日记录页 — 疼痛热力标注

- **人体疼痛热力标注**：在人体图上标注疼痛部位
- **多部位疼痛记录**：支持记录多个部位的疼痛类型和等级（1-5 级）
- **体重与 BMI 跟踪**：记录当日体重和自动计算的 BMI
- **情绪状态记录**：5 星评分记录当日情绪
- **记录编辑**：支持编辑最近一条记录

### 健康数据分析页

- **趋势图表**：基于 Chart.js 的体重、BMI 趋势折线图
- **疼痛分布统计**：疼痛部位分布和频次分析
- **时间筛选**：支持近 7 天 / 近 30 天数据切换

### 体检报告档案页

- **报告上传**：支持上传 PDF 和图片格式的体检报告
- **异常指标标记**：在报告上标记和备注异常指标
- **档案管理**：按时间线管理历次体检报告

### 历史记录页

- **日记列表**：以卡片形式展示所有身体日记记录
- **日期筛选**：支持按全部 / 本周 / 本月筛选
- **记录详情**：点击查看单条记录的完整信息

## 更新日志

### v2.0 — 手绘笔记风格重构（2025-07-12）

**视觉设计全面升级**

- 全局视觉风格从扁平现代风重构为手绘笔记风格
- 引入 Caveat 手写字体和 Nunito 正文字体
- 背景改为米白色噪点纹理，叠加手绘草图辅助线、箭头标注等装饰元素
- 卡片改为便签风格：深色描边、硬阴影、横线纸纹理，随机分配四种胶带颜色（紫色/薄荷/粉色/橙色）
- 按钮、标签、弹窗、输入框、滑块、复选框、星星评分等组件全部适配手绘风格
- 配色体系从青绿色系(#4ECDC4)整体切换为绿色(#22C55E)、橙色(#F97316)、蓝色(#3B82F6)新色板

**首页交互闭环**

- 首页布局从"标题+人物+功能卡片"重构为中心辐射式身体地图布局
- 新增身体地图可视化层：彩色健康圆点分布、手绘连接曲线、风险警告三角，均带动画效果
- 新增"选症状→点部位→生成记录"的首页快速记录交互闭环
- 新增症状多选切换逻辑（酸痛/紧绷/刺痛/乏力），选中后点亮全身点位
- 新增临时记录卡片的渲染、删除和提交功能
- 新增无症状选中时点击点位展示部位详情浮窗
- 新增身体部位详情数据映射（头部/肩颈/胸腹/腰/四肢）

### v1.0 — 初始版本

- 完成身体日记前端 Demo 的基础功能开发
- 包含首页、今日记录、健康分析、体检档案、历史记录五个页面
- 实现基于 localStorage 的本地数据存储
- 采用 Chart.js 实现数据可视化图表

## 技术栈

- **HTML5 / CSS3 / JavaScript**：纯前端单页应用
- **Chart.js**：数据可视化图表
- **SVG**：身体地图可视化与人物绘制
- **localStorage**：本地数据持久化
- **Google Fonts**：Caveat（手写体）+ Nunito（正文字体）

## 快速开始

直接在浏览器中打开 `body-map-demo.html` 即可运行，无需安装任何依赖。

```bash
# 克隆仓库
git clone https://github.com/fStardust/BodyMap.git

# 进入目录
cd BodyMap

# 用浏览器打开 Demo
open body-map-demo.html
```
